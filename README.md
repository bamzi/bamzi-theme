###### Install

`Plug "bamzi/bamzi-theme"`
or your choice of package manager

To set the theme you can use the following.

```lua
vim.cmd[[colorscheme bamzi]]
```

### Generator

`cargo install colorgen-nvim`

`colorgen-nvim bamzi.toml`
